# DF-RollerBoard
Some Arduino Projects for Digital Physics. Моделирование колебаний доски на вращающихся валиках
Численное моделирование задачи о движении доски, лежащей на вращающихся роликах. Постановка задачи, алгоритм решения и результаты расчетов - в файле Descriptions4.pdf. Проведен анализ накопления погрешностей при выполнении расчетов по рекуррентным формулам.
